# 📖 Project Overview

**Pulse-AI** is a simple **Rule-Based Artificial Intelligence Chatbot** built using Python. The chatbot responds to predefined user inputs such as greetings, questions, jokes, facts, and farewell messages.

Unlike machine learning chatbots, Pulse-AI uses a **dictionary-based rule system** to generate responses, making it an excellent beginner project for understanding chatbot logic and conversational AI.

# ✨ Features

- 🤖 Interactive command-line chatbot
- 💬 Responds to greetings
- ❓ Answers common questions
- 😂 Tells programming jokes
- 📚 Shares interesting facts
- 🌞 Greets users according to the time of day
- ❤️ Responds to thank-you messages
- 🚪 Exit functionality using `exit` or `quit`
- ⚠️ Handles unknown inputs gracefully
- 🧹 Ignores empty user input

# 🚀 How It Works

```text
Start Program
      │
      ▼
Display Welcome Message
      │
      ▼
Wait for User Input
      │
      ▼
Check User Input
      │
      ▼
Find Matching Response
      │
      ▼
Display Reply
      │
      ▼
Repeat Until User Types Exit
```
